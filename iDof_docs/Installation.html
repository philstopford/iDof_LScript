iDof can be installed as a plugin.  In LW 8.5, that entails:
    Utilities Tab (at the top) >> Add Plugins
 
Or it can be run directly from the Processing Tab:
    Window pulldown (upper left) >> Image Processing
 
If running directly, select Lscript/RT from the Add Image Filter pulldown and locate the iDof Channels and/or iDof Post Blur files on your local disk.
 
If running after using Add Plugins, select iDof Channels and/or iDof Post Blur from the Add Image Filter pulldown.
